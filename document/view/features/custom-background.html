# 自定义背景图片

  • 具体操作步骤
    • 打开原神启动器Plus，在设置中勾选[不使用网络背景]
    • 在程序设置中找到[主页背景设置]的按钮，点击后选择你想要使用的图片即可
    • 注意：支持的图片格式为png、jpg或webp，其他格式的图片未支持
